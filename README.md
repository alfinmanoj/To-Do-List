# 📝 To-Do List App

This is my first project using HTML, CSS, and JavaScript.

## 🚀 Features

* Add tasks
* Mark tasks as complete
* Delete tasks
* Press Enter to add task

## 🛠️ Tech Used

* HTML
* CSS
* JavaScript

## 📌 Future Improvements

* Save tasks using localStorage
* Edit tasks
* Add filters

## ▶️ How to Run

Open `index.html` in your browser
